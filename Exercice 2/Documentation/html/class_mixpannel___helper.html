Detailed Description

The Mixpannel_Helper is dedicated to simplification of Mixpanel tool integration.

It Handle a generic implementation of launch counter, first launch event and party counter it also simplify GamePlayed and appLaunched event sending by reduce the integration to only two calls void GamePlayedEvent(int score) and void GameLaunchedEvent() nothing more have to be written by gameDevs.

Parameters
AppName,Appdescriptor for Mixpanel UI and sort options
verbose,usefullfor event debugging
